# Arrays Problems# Arrays

This directory contains my solutions to Array-based problems from LeetCode.

Arrays are one of the most fundamental data structures and are frequently tested in coding interviews. These problems focus on improving problem-solving skills involving traversal, searching, sorting, hashing, prefix sums, two pointers, and sliding window techniques.

---

## Concepts Covered

- Array Traversal
- Hash Maps
- Prefix Sum
- Two Pointers
- Sliding Window
- Sorting
- Binary Search on Arrays
- Greedy Techniques

---

## Problems Solved

| Problem | Difficulty | Technique |
|----------|------------|------------|
| Two Sum | Easy | Hash Map |
| Contains Duplicate | Easy | Hash Set |
| Best Time to Buy and Sell Stock | Easy | Greedy |
| Product of Array Except Self | Medium | Prefix & Suffix Arrays |
| Maximum Subarray | Medium | Kadane's Algorithm |
| Move Zeroes | Easy | Two Pointers |
| Merge Sorted Array | Easy | Two Pointers |
| Majority Element | Easy | Hashing |
